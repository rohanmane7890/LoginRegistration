# Login Registration 
Login & Registration System
A simple Java Web Application built using JSP, Servlets, JDBC, and MySQL.
This project demonstrates user registration, login, search, and session handling

# Features
User Registration
User Login & Logout
Search Functionality
JSP-based UI Pages
MySQL Database Connectivity (JDBC)
Session Management
Basic validation & error handling

# Technologies Used
Java (JDK 8+)
JSP & Servlets
JDBC
MySQL
Apache Tomcat (9/10)
HTML, CSS
Eclipse IDE
Git & GitHub


# How to run the project 
Clone the repository:
git clone https://github.com/your-username/your-repo-name.git

# Open Eclipse
File → Import → Existing Projects into Workspace
Select project folder
Configure Apache Tomcat Server
Import MySQL JDBC Driver (Connector/J)
Run the project:
Right-click project → Run on Server
